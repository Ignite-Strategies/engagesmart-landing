Security at EngageSmart

Transport & storage

HTTPS/TLS for data in transit; tokens and other sensitive secrets encrypted at rest.

Access control

Role-based, least-privilege access; admin actions logged and reviewed.

Reliability

Regular backups and tested recovery procedures.

Vulnerability reporting

If you discover a security vulnerability, please report it to:

security@engagesmart.app

We take all reports seriously and will respond promptly.

Incidents

We aim to notify impacted users and (where required) regulators within 72 hours of becoming aware of a data breach.
